This is NetRadiant and Q3Map2 for Windows 32-bit and Linux 64-bit compatible with Warsow 2.0.

Built from a source snapshot from November 15, 2015:
https://gitlab.com/xonotic/netradiant/commit/c51c744b5ff06a6553a23914c274d6a2911c8eda

Contains the Warsow game pack:
https://github.com/Warsow/NetRadiantPack

To install other game packs, refer to the shell script:
https://gitlab.com/xonotic/netradiant/blob/master/download-gamepacks.sh
